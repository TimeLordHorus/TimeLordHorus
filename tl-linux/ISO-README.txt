Q: Can I dual-boot with Windows?
A: Yes! The installer can detect Windows and set up dual-boot.

Q: Will this work on older computers?
A: Yes! TL Linux uses XFCE, which is lightweight and runs well on
   older hardware. Minimum: 2GB RAM, any 64-bit processor.

Q: Can I install additional software?
A: Absolutely! Use apt package manager or install from .deb files.
   Example: sudo apt install gimp

Q: How do I update the system?
A: Updates are automatic and safe thanks to A/B partitions. Manual:
   sudo apt update && sudo apt upgrade

Q: Is my data safe?
A: Very! The dual-partition system means updates never overwrite
   your working system. Plus automatic backups protect user data.

Q: Can I use this without accessibility features?
A: Of course! TL Linux is a full-featured Linux distribution.
   Accessibility features are optional but always available.
